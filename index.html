Project: Sound Meter

By Juan David Ramírez, Pedro Pablo Solano and Nicolas Romero • Of 9B • In 2025

The "Sound Meter" project seeks to reduce noise in hallways through a system that measures sound and activates lights that indicate the ambient volume level.

Show process

Objective of the project

Design a system with sound sensors and LED lights that maintains peace and quiet in school spaces by providing visual alerts when noise exceeds certain levels.

Components and Materials

  • Sound Sensor
  • Sensor PIR
  • LED red, LED yellow and LED green
  • ESP32
  • Cables
  • Silicone
  • Electrical tape
  • 3D printing

Development process

  1. Identify the problem

    We detect that in the hallway in front of the technology classroom were a lot of sound when the childrens passed by, and it affect the class all the time.

  2. System design

    We proposed to create a functional circuit that depending on what the sound sensor detects, shows through LED lights, which would be: green LED when the volume is low, yellow LED when it is moderate and red LED when it is high.

  3. Problems on the design

    The teacher made us see that the children were not gonna pay attention to us with just a few little lights, and in addition, our project was missing 1 actuator and a sensor.

  4. Solutions

    We added an LCD screen to show the status of the LEDs, and so that it was not so intuitive, we also added a PIR sensor so that when it does not detect movement it does not say the volume status, and finally a mini speaker in which the voice of someone is recorded to say "please be quiet", and thus try to raise awareness.

  5. Web page

    We decided to create a website for the project so we could see the number of times there are shouts per day and week, and then compare weeks or days to see if the project is working or not.

  6. Programming

    We told Chat GPT about the whole project and asked him to make us a working code, and fortunately it worked on the first attempts.

  7. 3D disgn

    For better precision, we created the entire box for the project in Tinkercad so we could print it later. This box helped us present the project without adding much.

  8. Construction

    After having the circuit with the code working, we began to introduce it to the box, with the help of insulating tape and silicone, so that the result would not look ugly, and we could present it to you.

  9. Tests and adjustments

    Tests were conducted in the classroom to calibrate the sensor sensitivity and the response time of the LEDs and speaker.

Results

The system worked correctly, turning on the LEDs depending on the noise level, and the speaker if the noise was too loud. It raised some awareness among the children who were presented with the project on Science Day. Below is a table showing how the project works.

  • The project helped raise awareness among students about noise control on Science Day.
  • A functional system was implemented with basic electronic components.
